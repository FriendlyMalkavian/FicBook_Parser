FicBook Parser v0.11 — Windows GUI

Графическое приложение для скачивания фанфиков с ficbook.net.

Системные требования:
- Windows 10 / Windows 11
- 64-разрядная система

Файлы:
  dist\FicBookParser.exe  — готовый к запуску исполняемый файл
  ficbook_gui.py          — исходный код (Python 3.10+)

Форматы сохранения:
  TXT, MD, JSON, EPUB, FB2

Файлы сохраняются в папку "Загрузки" (Downloads).
